## 数字各位数求和
> 第二章 中期练习题 中等 第3题

### 题目
#### 简介
实现一个函数`sum_digits(num)`，
接受一个整数串`num`作为参数。
返回该数各位数字的和。

#### 详细说明
`1234`的各位数字为
- 个位：4
- 十位：3
- 百位：2
- 千位：1

所以`1234`各位数字之和为 4+3+2+1=10

同理`987`各位数字之和为 7+8+9=24

#### 示例
示例一
```python
print(sum_digits(1234))
```
输出为
```python
10
```
示例二
```python
print(sum_digits(9999))
```
输出为
```python
36
```

### 答案
```python
def sum_digits(num):
    s = 0
    while num > 0:
        last = num % 10
        s += last
        num = num // 10

    return s
```
